Thư mục này chứa tài liệu, hướng dẫn, và blueprint cho các workflow n8n.

## File Có Sẵn

- `sync-podcast-workflow.md` - Hướng dẫn workflow podcast
- `sync-youtube-workflow.md` - Hướng dẫn workflow YouTube

1. Mở tài liệu tương ứng
